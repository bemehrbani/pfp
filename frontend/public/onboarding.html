Admin Guide

System administrators with full platform access

🔑 First Login

  1. Navigate to http://65.109.198.200:8080
  2. Log in with the admin credentials provided by the system operator
  3. You'll land on the Dashboard showing platform-wide statistics

⚡ Key Capabilities

FeatureWhere to Find It
User managementDjango Admin → Users
All campaigns overviewCampaigns page
System analyticsAnalytics → System
Telegram bot configDjango Admin → Telegram settings
Database managementDjango Admin (/admin/)

🛠️ Setting Up the Platform

  1. Create Campaign Managers — Django Admin → Users → Add User → Set role to Campaign Manager
  2. Configure Telegram Bot — Set the TELEGRAM_BOT_TOKEN environment variable
  3. Review System Health — Visit /api/health/ to verify all services are operational
  4. Monitor Analytics — Dashboard shows active campaigns, user growth, and task completion rates
Django Admin: Access the admin panel at /admin/ for direct database management — create users, manage campaigns, view activity logs, and configure system settings.

Campaign Manager Guide

Create campaigns, assign tasks, and coordinate volunteers

🚀 Getting Started

  1. Log in at http://65.109.198.200:8080
  2. Your Dashboard shows your campaigns, task progress, and volunteer activity
  3. Click Campaigns in the sidebar to manage your campaigns

📋 Creating a Campaign

  1. Go to Campaigns → Create Campaign
  2. Fill in: Name (clear, compelling title), Description (campaign goals), Category, Start/End Dates, Targets (member, activity, social post targets), and Region
  3. Click Create to publish

✅ Managing Tasks

  1. Navigate to your campaign → Tasks tab
  2. Click Create Task to add work items — set Title, Task Type (Social Media, Outreach, etc.), Points, Priority, and Deadline
  3. Tasks appear in the volunteer's task list once assigned

📊 Tracking Progress

📈Campaign Analytics — Click any campaign → Analytics tab for member recruitment, completion rates, top volunteers, and task distribution
🎯Dashboard — Quick overview of all your campaigns' health
👥Volunteer Coordination — Review submitted proofs, award points upon verification, monitor real-time activity
🟢

Volunteer Guide

Browse campaigns, complete tasks, and earn points

👋 Getting Started

  1. Log in at http://65.109.198.200:8080
  2. Your Dashboard shows My Tasks, Points Earned, and Active Campaigns

🤝 Joining a Campaign

  1. Go to Campaigns in the sidebar
  2. Browse available campaigns — click on one to see its details, tasks, and progress
  3. Click Join Campaign to become a volunteer

📝 Completing Tasks

  1. Go to Tasks in the sidebar — you'll see Available, In Progress, and Completed tabs
  2. Click on a task and press Accept Task to start working on it
  3. Once complete, click Submit Proof and provide evidence (photo, link, or description)
  4. A Campaign Manager will verify your submission and award points

💡 Tips for Success

🎯Start small — begin with low-point tasks to learn the platform
⏰Check deadlines — tasks may expire if not completed in time
📸Provide clear proof — screenshots, links, or photos help managers verify quickly
🔥Stay active — campaign managers can see your activity history

🔐 Authentication

Send a POST request with your credentials to receive a JWT token:

POST /api/auth/login/ Content-Type: application/json {"username": "your_user", "password": "your_pass"}

Include the token in subsequent requests:

Authorization: Bearer <your_token>
